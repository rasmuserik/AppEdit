AppEdit makes it fast to start coding HTML5/Apps.

In the top-bar above:
  • You can Read the source code / documentation for the current app.
  • You can Edit the source code of the current app, and see the result live.
  • You can run the App
  • You can publish the app, and Share the link that runs the current app

Features

Intended, not implemented yet
  • Live editing - when you change the source file, the result in the app will show immediately
  • No setup - just open the website, and you can start coding
  • Easy deployment - distribute your app as a website, as an Android app, as an iOS app, as a Facebook app, as a Chrome app, as a NPM package, ...
  • Learn from others - code are designed to be read, and experimented with. See how other apps works, and build upon them
